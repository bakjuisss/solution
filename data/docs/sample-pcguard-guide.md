# PCGuard 설치 가이드

## 개요

PCGuard는 엔드포인트 보안 솔루션으로, 관리 서버와 에이전트로 구성됩니다.

## 시스템 요구사항

- Windows Server 2016 이상 (관리 서버)
- Windows 10/11 (에이전트)
- .NET Framework 4.7.2 이상

## 설치 절차

1. 관리 콘솔 설치 파일(`PCGuardSetup.exe`)을 실행합니다.
2. 설치 마법사에서 설치 경로를 선택하고 **다음**을 클릭합니다.
3. 데이터베이스 연결 정보를 입력합니다. 기본 포트는 1433입니다.
4. 관리자 계정을 생성하고 초기 비밀번호를 설정합니다.
5. 설치 완료 후 웹 콘솔(`https://서버IP:8443`)에 접속합니다.

## 라이선스 등록

1. 웹 콘솔에 관리자로 로그인합니다.
2. **설정 > 라이선스 관리** 메뉴로 이동합니다.
3. 라이선스 키 파일(`.lic`)을 업로드하거나 키 문자열을 직접 입력합니다.
4. **등록** 버튼을 클릭하면 에이전트 수와 만료일이 표시됩니다.

## 에이전트 배포

- GPO를 통한 일괄 배포를 권장합니다.
- 수동 설치 시 `/silent` 옵션으로 무인 설치가 가능합니다.
- 에이전트는 관리 서버 주소를 레지스트리 `HKLM\Software\Plantynet\PCGuard\Server`에 저장합니다.

## 문제 해결

- 에이전트가 서버에 연결되지 않으면 방화벽에서 8443, 8080 포트를 허용하세요.
- 라이선스 오류 시 서버 시간이 NTP와 동기화되어 있는지 확인하세요.
